# Basic data types ( C )
### **C 语言格式说明符的所有功能总结**

格式说明符（Format Specifier）不仅用于 `printf`，也适用于 `scanf`、`sprintf`、`fprintf` 等函数。它们的主要功能包括 **占位、数据类型指定、对齐、宽度控制、精度控制、填充、数值进制转换、科学计数法等**。

---

## **1. 数据类型指定**
格式说明符用于指定不同类型的数据，确保正确的格式化输入/输出。

| **格式说明符** | **数据类型** | **说明** |
|--------------|------------|-------------------------------|
| `%d` / `%i` | `int` | 有符号整数（十进制） |
| `%u` | `unsigned int` | 无符号整数（十进制） |
| `%o` | `unsigned int` | 无符号八进制整数 |
| `%x` / `%X` | `unsigned int` | 无符号十六进制整数（x 小写，X 大写） |
| `%f` | `float` / `double` | 浮点数（小数点表示法） |
| `%e` / `%E` | `float` / `double` | 科学计数法（e 小写，E 大写） |
| `%g` / `%G` | `float` / `double` | 根据数值大小自动选择 `%f` 或 `%e` |
| `%c` | `char` | 单个字符 |
| `%s` | `char *` | 字符串 |
| `%p` | `void *` | 指针地址 |
| `%lld` | `long long` | 长长整数（64 位） |
| `%llu` | `unsigned long long` | 无符号长长整数（64 位） |
| `%Lf` | `long double` | 长双精度浮点数 |

---

## **2. 对齐与宽度控制**
用于控制数据的显示宽度以及对齐方式。

| **格式说明符** | **功能** | **示例** | **输出** |
|--------------|---------|--------|--------|
| `%5d` | 右对齐，占 5 个字符宽度 | `printf("%5d", 42);` | `   42` |
| `%-5d` | 左对齐，占 5 个字符宽度 | `printf("%-5d", 42);` | `42   ` |
| `%05d` | 右对齐，空位用 `0` 补齐 | `printf("%05d", 42);` | `00042` |

---

## **3. 精度控制**
用于控制浮点数的小数位数或字符串的最大字符数。

| **格式说明符** | **功能** | **示例** | **输出** |
|--------------|---------|--------|--------|
| `%.2f` | 保留 2 位小数 | `printf("%.2f", 3.14159);` | `3.14` |
| `%.5f` | 保留 5 位小数 | `printf("%.5f", 3.1);` | `3.10000` |
| `%.3s` | 只显示字符串的前 3 个字符 | `printf("%.3s", "Hello");` | `Hel` |

---

## **4. 数值进制转换**
用于在 **十进制、八进制、十六进制** 之间转换。

| **格式说明符** | **功能** | **示例** | **输出** |
|--------------|---------|--------|--------|
| `%d` | 十进制整数 | `printf("%d", 10);` | `10` |
| `%o` | 八进制整数 | `printf("%o", 10);` | `12` |
| `%x` | 十六进制（小写） | `printf("%x", 255);` | `ff` |
| `%X` | 十六进制（大写） | `printf("%X", 255);` | `FF` |

---

## **5. 科学计数法**
用于表示非常大的或非常小的数值。

| **格式说明符** | **功能** | **示例** | **输出** |
|--------------|---------|--------|--------|
| `%e` | 科学计数法（小写） | `printf("%e", 123.456);` | `1.234560e+02` |
| `%E` | 科学计数法（大写） | `printf("%E", 123.456);` | `1.234560E+02` |
| `%.3e` | 保留 3 位小数的科学计数法 | `printf("%.3e", 0.0004567);` | `4.567e-04` |

---

## **6. 动态宽度与精度**
`*` 允许动态指定宽度和精度。

| **格式说明符** | **功能** | **示例** | **输出** |
|--------------|---------|--------|--------|
| `%*d` | 动态指定宽度 | `printf("%*d", 6, 42);` | `    42` |
| `%.*f` | 动态指定小数位数 | `printf("%.*f", 3, 3.14159);` | `3.142` |

---

## **7. 指针地址格式**
`%p` 用于输出指针变量的地址。

```c
#include <stdio.h>

int main() {
    int a = 10;
    printf("Address: %p\n", (void*)&a);
    return 0;
}
```
**输出示例：**
```
Address: 0x7ffcc3a4d6f8
```

---

## **8. 千位分隔符（非标准扩展）**
部分编译器（如 GNU C）支持 `%'d` 进行 **千位分隔**：

```c
printf("%'d\n", 1000000);
```
**可能输出：**
```
1,000,000
```
**注意**：并非所有 C 语言实现都支持该格式。

---

## **9. 组合格式说明符**
可以组合 **宽度、对齐、精度** 来控制输出格式：

```c
printf("%10.2f\n", 3.14159);
```
**输出：**
```
      3.14
```
> - **`10.2f`**：右对齐，占 **10** 个字符宽度，小数点后保留 **2** 位。

---

## **10. `scanf` 相关的格式**
`scanf` 读取输入时，也可以使用格式说明符，支持：
- **忽略某些输入**
- **限制字符串长度**
- **匹配特定模式**

| **格式说明符** | **功能** | **示例** |
|--------------|---------|--------|
| `%*d` | 跳过整数输入 | `scanf("%*d %d", &x);` |
| `%5s` | 读取最多 5 个字符 | `scanf("%5s", str);` |
| `%[0-9]` | 只读取数字 | `scanf("%[0-9]", str);` |

---

## **总结**
| **功能类别** | **示例** | **说明** |
|------------|---------|---------|
| **数据类型指定** | `%d, %f, %s, %p` | 指定整数、浮点数、字符串、指针等类型 |
| **对齐** | `%5d, %-5d` | 右对齐、左对齐 |
| **精度控制** | `%.2f, %.3s` | 小数点位数、字符串长度 |
| **进制转换** | `%o, %x, %X` | 八进制、十六进制 |
| **科学计数法** | `%e, %E` | 科学计数法 |
| **动态宽度** | `%*d, %.*f` | 宽度、小数位数由变量决定 |
| **指针地址** | `%p` | 输出指针地址 |
| **输入控制** | `%*d, %[0-9]` | `scanf` 相关功能 |
